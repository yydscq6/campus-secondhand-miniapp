# 校易淘 — 校园二手交易小程序

> 基于微信云开发的校园二手交易平台，采用紫罗兰渐变主题设计，支持**校园身份认证**、**校区精准推荐**、商品浏览、发布、交易、评价、议价等完整功能。

## 技术栈

| 类型 | 技术 |
|------|------|
| 前端框架 | 微信小程序原生开发 |
| 后端服务 | 微信云开发（云函数 + 云数据库） |
| 设计风格 | 紫罗兰渐变 + 毛玻璃 + 微交互动画 |
| 图标方案 | 自研 SVG 线性图标（37枚） |
| 数据层 | Mock 数据（预留云函数接口） |

## 设计规范

- **主题色**: `#6C5CE7` → `#a29bfe` → `#74b9ff`（渐变）
- **圆角**: 24rpx（卡片）/ 16rpx（小元素）/ 32rpx（大区域）
- **阴影**: 紫色系投影 `rgba(108,92,231,0.08)`
- **动画**: fadeIn / slideUp / scale 弹性过渡
- **图标**: 24×24 viewBox，2px 描边，圆角端点，线性风格

## 核心亮点

### 🎓 校园身份认证

使用**学号 + 校园邮箱验证码**完成实名认证，确保每位用户均为真实在校学生，解决二手交易中的信任问题。

- 支持全国 **9 个城市、36 所高校**（北京/上海/广州/南京/武汉/杭州/成都/西安/长沙）
- 邮箱后缀自动校验（如 `@pku.edu.cn`、`@hutb.edu.cn`）
- 认证信息本地持久化，退出登录后状态保留

### 📍 校区精准推荐

认证完成后，首页自动切换为**"本校推荐"模式**，仅展示本校区商品，也可一键切换查看全部校区。

- 首页、分类页均支持校区筛选切换
- 商品卡片标注来源校区（📍北京大学）
- 发布商品时自动填充认证校区

## 页面结构（13 个页面）

```
pages/
├── index/          # 首页（轮播 + 分类 + 校区筛选 + 热门 + 最新）

├── chat/           # 聊天（买家卖家对话）
└── review/         # 评价（查看 / 发表评价）
```

## 图标系统（37 枚 SVG）

所有图标采用统一线性风格：24×24 viewBox / 2px 描边 / 圆角端点 / `currentColor` 继承文字色。

| 类型 | 图标名称 |
|------|---------|
| TabBar (8) | `tab-home` `tab-category` `tab-publish` `tab-mine`（各含 `-active` 版本） |
| 导航 (4) | `arrow-left` `arrow-right` `close` `search` |
| 功能 (8) | `bell` `heart` `heart-fill` `message` `shopping-bag` `eye` `edit` `trash` |
| 保障 (5) | `shield` `shield-check` `check-circle` `zap` `refresh-cw` |
| 分类 (6) | `smartphone` `book-open` `fire` `star` `star-fill` `tag` |
| 菜单 (7) | `user` `package` `clipboard` `settings` `clock` `lock` `tag` |

TabBar 图标使用 Pillow（Python）生成 81×81 RGBA PNG，页面内图标使用 SVG `<image>` 引用。

## 使用方法

### 1. 克隆项目
```bash
git clone https://github.com/yydscq6/campus-secondhand-miniapp.git
```

### 2. 导入微信开发者工具
1. 打开 **微信开发者工具**
2. 选择 **导入项目**
3. 项目目录选择 `campus-secondhand-miniapp` 文件夹
4. AppID 填写：`wx56b37bc03e307702`
5. 后端服务选择 **微信云开发**
6. 点击 **编译** 即可预览
